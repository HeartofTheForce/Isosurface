# Isosurface

A highly optimized C++ OpenGL implementation of the Dual Contouring Isosurface extraction algorithm, designed to operate on uniform grids in parallel.

# Built using

- C++ 11
- CMAKE
- OpenMP
- OpenGL 3.3
- GLM

# Getting Start

- `cmake . -B build`
- `cmake --build build`
- `build/out/main.exe`
